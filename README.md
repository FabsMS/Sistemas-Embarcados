# Sistemas Embarcados

Este repositório foi criado para armazenar as atividades desenvolvidas ao longo da disciplina de **Sistemas Embarcados**.

Aqui serão reunidos exercícios, projetos, experimentos e demais materiais produzidos durante o curso. O objetivo é centralizar o desenvolvimento das atividades em um único local, facilitando a organização, o versionamento e o acompanhamento da evolução ao longo da disciplina.

## Organização

As atividades estarão organizadas em pastas separadas, de acordo com o tema, aula ou projeto correspondente.

Exemplo de estrutura:

```
/atividade-01
/atividade-02
/atividade-N
/projeto-X
/projeto-Y
```

Cada pasta pode conter códigos, diagramas, documentação ou quaisquer outros arquivos relacionados à atividade.

## Observação

Este repositório tem caráter **acadêmico**, sendo utilizado exclusivamente para fins de estudo e registro das atividades desenvolvidas durante a disciplina.
